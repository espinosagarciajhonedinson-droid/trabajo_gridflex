# Layout con Grid, Flexbox y Float

Un layout sencillo que demuestra cómo combinar tres técnicas diferentes de CSS:

## Características

- **Grid:** Estructura principal con 3 columnas
- **Flexbox:** Navegación, sidebars y footer
- **Float:** Imagen que envuelve el contenido de texto

## Estructura

```
trabajo_flex-grid-float/
├── index.html
├── styles.css
└── README.md
```

## Cómo usar

1. Abre `index.html` en tu navegador
2. O sirve los archivos con un servidor local:

```bash
python3 -m http.server 8000
```

Luego accede a `http://localhost:8000`

## Responsivo

El layout es completamente responsivo y se adapta a dispositivos móviles.

## Tecnologías

- HTML5
- CSS3 (Grid, Flexbox, Float)
